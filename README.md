# DevOps Home Lab (Proxmox + Ubuntu + Terraform)

## Overview

This project documents the build of a personal DevOps lab environment using:

* Proxmox (Type 1 Hypervisor)
* Ubuntu Server (Virtual Machines)
* Terraform (Infrastructure as Code)

The goal is to simulate real-world infrastructure and practice DevOps workflows without using a personal laptop.

---

## Architecture

Laptop → SSH → Ubuntu VM → Proxmox → Physical Server

---

## Project Structure

* `/docs` → Full setup guides
* `/proxmox` → Hypervisor setup
* `/ubuntu` → VM configuration
* `/terraform` → Infrastructure as Code labs
